概述

JavaScript 是一种高层次、动态类型的编程语言，广泛用于Web开发，特别是在前端开发中。它是浏览器支持的核心语言之一，也可以用于服务器端（如Node.js）。

基础语法

变量声明

  • var: 变量声明，存在函数作用域。
  • let: 变量声明，存在块级作用域。
  • const: 常量声明，值不可变，存在块级作用域。

数据类型

  • 原始数据类型：undefined, null, boolean, number, string, symbol（ES6+）
  • 引用数据类型：Object, Array, Function

运算符

  • 算术运算符：+, -, *, /, %, ++, --
  • 比较运算符：==, ===, !=, !==, >, <, >=, <=
  • 逻辑运算符：&&, ||, !
  • 三元运算符：condition ? value1 : value2
  • 赋值运算符：=, +=, -=, *=, /=

控制流

  • 条件语句：if, else, else if, switch
  • 循环语句：for, while, do...while, for...in, for...of
